Array
Array, data type objectga kiradi. Arrayni ikkinchi nomi massiv hisoblanadi. Array da sonlar indexsi 0 dan boshlib cheksizlikgacha bolishi mumkin. Masalan: ["Anna",9,true,NaN] bu yerda "Anna" - 0 inchi indexda turibdi va hokazo. Arrayni yaratishni to'rta usuli bor. Inshalaysni ichiga value berib qo'yish (defoult), Inshalys yozib pastidan indexlarini olib value berish, Inshalayz dan keyin "new Array(...)" kalit sozini kiritib Array hosil qilsa bo'ladi va oxirgi usul "Array(...)" kalit sozi orqali Inshalasyzdan keyin ichiga bersa bo'ladi. Eslatib o'tamiz Arrayni ikkinchi nomi Massiv deyiladi. Massivni indexlarini olib ishlatish ham mumkin. Masalan: consol.log(arrayname(0,1,2,3,....,infinity)). Albatta birinchi Massiv hosil qilish kere boladi. Xo'p, yana Arrayda "length" = uzunlik ma'nosini bildiradi. Bu Massivni uzunligini olishga ishlatiladi. Masalan: bir inshalaysga massiv hosil qilib uni yana bitta inshalaysga arrayningnomi.length qilsa bo'ladi. Yanna bitta narsa agar lenght qilsangiz sizg o'sha massivni indexsiga biita keyin oladi, yani bitta index ko'p oladi. Masalan: massiv uzunligi 5 bo'lsa, 6 gacha oladi. Shuning uchun ko'pincha massivni ununligni consolga chiqarayotganda console.log((uzunligini qaysidir inshalayzga olingan)o'zgaruvchi-1) qilinadi. Chunki agar uzunligidan birni ayirsangiz shunda to'gri bo'ladi, chunki lenght massivni bitta ko'p index oladi.
Arraydni o'zgartirish ham mumkin. Array ochib olingandan keyin Arrayni nomini kiritib nechinchi indexsini ochirmoqchiligingizni yozasiz va teng qilib o'zgartirmoqchi bo'lgan valueni kiritasiz. Keyin osha o'zgaruvchini olib consolga bersangiz o'zgartirgan valuengiz chiqadi. Arrayga yana value qo'shsa bo'ladi. Massiv o'zgaruvchiga olingan. Massiv qo'shishni ikkita usuli bor birinchisi o'zgaruvshiga olib arrayname[arrayname.lenght] = value qilasiz shunda sizga oxirgi indexsiga value qo'shadi. Ikkinchi usuli o'zgaruvchiga olib jami nechta index bo'lsa shundan bitta keyingi indexsini olasiz ya'ni" arrayname[indexsi 5 bo'lsa, 6 kiritasiz. 10 bo'lsa, 11 kiritasiz va hokazo]
Arrayni o'chirsa ham bo'ladi. Array o'zgaruvchiga hosil qilingan. "delete" = o'chirmoq degani. delete arrayname[indexsi] shunda qaysi index bersangiz o'sha indexsini o'ciradi
